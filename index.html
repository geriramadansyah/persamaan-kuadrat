Persamaan Kuadrat Contoh Soal dan Penyelesaian

Persamaan kuadrat adalah persamaan polinomial atau suku banyak yang pangkat tertingginya adalah dua. Bentuk umum persamaan kuadrat adalah \(ax^{2}+bx+c=0\), dengan: a koefisien kuadrat dari x, b koefisien linear dari x, c konstanta, x variabel. 

Persamaan kuadrat dapat diselesaikan dengan beberapa cara, yaitu:

- Faktorisasi, yaitu mencari nilai yang jika dikalikan akan menghasilkan nilai lain
- Melengkapkan kuadrat sempurna, yaitu menyelesaikan persamaan kuadrat dengan melengkapkan kuadratnya sehingga menjadi sempurna
- Rumus kuadrat atau rumus ABC

1. Persamaan Kuadrat dengan konstanta di sisi kanan (ax² + bx + c = n)

Contoh Soal:

2x² + 4x - 6 = 10

Penyelesaian:

Langkah pertama adalah menyusun persamaan menjadi bentuk standar:
2x² + 4x - 6 - 10 = 0
2x² + 4x - 16 = 0
Kemudian bagi seluruh persamaan dengan 2:
x² + 2x - 8 = 0
Menggunakan rumus kuadrat:
x = (-b ± √(b² - 4ac)) / 2a

Dengan nilai a = 1, b = 2, c = -8:
x = (-2 ± √(2² - 4(1)(-8))) / 2(1)
x = (-2 ± √(4 + 32)) / 2
x = (-2 ± √36) / 2
x = (-2 ± 6) / 2
Sehingga, solusi untuk x adalah:
x = 2 atau x = -4

2. Persamaan Kuadrat dengan bentuk ax² + bx - c = n

Contoh Soal:

3x² + 5x - 2 = 7

Penyelesaian:

Langkah pertama adalah menyusun persamaan menjadi bentuk standar:
3x² + 5x - 2 - 7 = 0
3x² + 5x - 9 = 0

Menggunakan rumus kuadrat:
x = (-b ± √(b² - 4ac)) / 2a

Dengan nilai a = 3, b = 5, c = -9:
x = (-5 ± √(5² - 4(3)(-9))) / 2(3)
x = (-5 ± √(25 + 108)) / 6
x = (-5 ± √133) / 6
Hasilnya adalah:
x = (-5 ± √133) / 6

3. Persamaan Kuadrat dengan bentuk ax² - bx + c = n

Contoh Soal:

4x² - 6x + 5 = 3

Penyelesaian:

Langkah pertama adalah menyusun persamaan menjadi bentuk standar:
4x² - 6x + 5 - 3 = 0
4x² - 6x + 2 = 0

Menggunakan rumus kuadrat:
x = (-b ± √(b² - 4ac)) / 2a

Dengan nilai a = 4, b = -6, c = 2:
x = (6 ± √((-6)² - 4(4)(2))) / 8
x = (6 ± √(36 - 32)) / 8
x = (6 ± √4) / 8
x = (6 ± 2) / 8
Sehingga solusi untuk x adalah:
x = 1 atau x = 1/2

4. Persamaan Kuadrat dengan ketidaksetaraan ax² + bx + c ≤ n

Contoh Soal:

x² + 4x + 3 ≤ 0

Penyelesaian:

Faktorkan persamaan:
(x + 1)(x + 3) ≤ 0
Tentukan titik-titik kritis: x = -1 dan x = -3.
Uji tanda untuk interval (-∞, -3), (-3, -1), dan (-1, ∞):
- Untuk x = -4 (di interval (-∞, -3)): hasilnya positif.
- Untuk x = -2 (di interval (-3, -1)): hasilnya negatif.
- Untuk x = 0 (di interval (-1, ∞)): hasilnya positif.
Karena kita mencari nilai yang memenuhi ketidaksetaraan, maka solusi adalah:
-3 ≤ x ≤ -1

5. Persamaan Kuadrat dengan ketidaksetaraan ax² + bx + c ≥ n

Contoh Soal:

x² - 2x + 1 ≥ 0

Penyelesaian:

Faktorkan persamaan:
(x - 1)² ≥ 0
Karena kuadrat dari setiap bilangan selalu lebih besar atau sama dengan nol, maka ketidaksetaraan ini berlaku untuk semua nilai x:
Solusi: Semua nilai x ∈ ℝ

6. Persamaan Kuadrat dengan bentuk x² – nx > n

Contoh Soal:

x² - 6x > 8

Penyelesaian:

Langkah pertama adalah menyusun persamaan menjadi bentuk standar:
x² - 6x - 8 > 0
Faktorkan persamaan:
(x - 4)(x + 2) > 0
Tentukan titik-titik kritis: x = 4 dan x = -2.
Uji tanda untuk interval (-∞, -2), (-2, 4), dan (4, ∞):
- Untuk x = -3 (di interval (-∞, -2)): hasilnya positif.
- Untuk x = 0 (di interval (-2, 4)): hasilnya negatif.
- Untuk x = 5 (di interval (4, ∞)): hasilnya positif.
Solusi: x < -2 atau x > 4

7. Persamaan Kuadrat dengan bentuk x² – n > n

Contoh Soal:

x² - 4 > 5

Penyelesaian:

Langkah pertama adalah menyusun persamaan menjadi bentuk standar:
x² - 4 - 5 > 0
x² - 9 > 0
Faktorkan persamaan:
(x - 3)(x + 3) > 0
Tentukan titik-titik kritis: x = 3 dan x = -3.
Uji tanda untuk interval (-∞, -3), (-3, 3), dan (3, ∞):
- Untuk x = -4 (di interval (-∞, -3)): hasilnya positif.
- Untuk x = 0 (di interval (-3, 3)): hasilnya negatif.
- Untuk x = 4 (di interval (3, ∞)): hasilnya positif.
Solusi: x < -3 atau x > 3
